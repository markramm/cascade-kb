Embeddable timelines

Every curated timeline has an iframe-ready version stripped of site chrome. Drop one in a Substack post, blog article, or static page.

How to embed

Each timeline preset has an /embed/ URL that renders the chart without site header or footer. Use it inside an <iframe>: 

<iframe
  src="https://capturecascade.org/embed/timelines/project-2025-architects/"
  width="100%" height="780"
  style="border:1px solid #d9d2bf;"
  title="The Project 2025 architects — capturecascade.org">
</iframe>

The embedded chart is fully interactive — hover any marker for date and lane, click to open the source event. The bottom strip credits capturecascade.org and links back.
