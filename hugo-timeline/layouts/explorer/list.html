How this works

Every actor in the corpus has a pre-computed event index that loads once when the page opens (~1.3 MB gzipped). Filter changes redraw the chart locally — no server round-trips, no telemetry.

Markers are sized by editorial importance (1–10 scale) and colored by the first capture lane on each event. Dense rows scale down marker size proportionally so 1,000+ events still render legibly. Click any marker to open its full record.

The URL state is canonical — copying the address bar gives anyone the exact same view. Use it as a citation in articles, footnotes, or social posts.
